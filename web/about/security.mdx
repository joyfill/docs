# Security & Support

Learn about our security practices and how to get help with the Web SDK.

## Security

### Data Encryption

All data transmitted between your app and Joyfill servers is encrypted using TLS 1.2 or higher.

```swift
// SDK uses secure HTTPS connections by default
Joyfill.configure(accessToken: "your_token")
```

### Access Tokens

Protect your access tokens and never commit them to version control:

```swift
// ❌ Don't do this
let token = "your_hardcoded_token"

// ✅ Do this instead
guard let token = ProcessInfo.processInfo.environment["JOYFILL_TOKEN"] else {
    fatalError("Missing token")
}
Joyfill.configure(accessToken: token)
```

### Data Storage

Form data is encrypted at rest when stored locally:

```swift
let config = JoyfillFormConfig(
    mode: .fill,
    templateId: "template_id"
)
config.encryptLocalStorage = true
```

### Sensitive Data

Mark fields containing sensitive information:

```swift
field.isSensitive = true // Data will be encrypted
```

## Best Practices

<AccordionGroup>
  <Accordion title="Token Management">
    - Store tokens in iOS Keychain
    - Rotate tokens regularly
    - Use environment-specific tokens
    - Never log tokens in production
  </Accordion>
  
  <Accordion title="Data Validation">
    - Always validate user input
    - Use schema validation
    - Implement custom validators
    - Sanitize data before submission
  </Accordion>
  
  <Accordion title="Error Handling">
    - Never expose sensitive error details to users
    - Log errors securely
    - Implement proper error recovery
    - Test error scenarios
  </Accordion>
</AccordionGroup>

## Compliance

Joyfill Web SDK is compliant with:
- GDPR (General Data Protection Regulation)
- CCPA (California Consumer Privacy Act)
- HIPAA (Healthcare Insurance Portability and Accountability Act)
- SOC 2 Type II

## Reporting Security Issues

If you discover a security vulnerability:

1. **DO NOT** open a public GitHub issue
2. Email security@joyfill.io with details
3. Include steps to reproduce if possible
4. We'll respond within 24 hours
