## Overview

This page provides sample projects demonstrating common patterns in JoyDoc. Each example is available in three different implementation styles: JS CDN, JS Module, and React.

## Sample Projects

Choose the integration method that best fits your project:

- **[JS CDN](./sample-projects-js-cdn)** - Quick integration using CDN script tags
- **[JS Module](./sample-projects-js-module)** - ES modules with build tools like Parcel
- **[React JS](./sample-projects-react)** - React v18 integration with components

Each guide includes complete examples, project structure, and configuration options.
