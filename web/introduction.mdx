# Web SDK Overview

Welcome to the Joyfill Web SDK documentation. Our SDK enables you to integrate powerful, customizable form experiences directly into your iOS applications.

## Key Features

<CardGroup cols={2}>
  <Card title="Easy Integration" icon="puzzle-piece">
    Simple SDK setup with JavaScript/TypeScript Package Manager or CocoaPods
  </Card>
  <Card title="Customizable UI" icon="palette">
    Fully customizable components to match your brand
  </Card>
  <Card title="Multiple Modes" icon="toggle-on">
    Support for readonly, fill, and edit modes
  </Card>
  <Card title="Event Handling" icon="bolt">
    Comprehensive event system for tracking user interactions
  </Card>
</CardGroup>
