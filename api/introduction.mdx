# Joyfill REST API

The Joyfill REST API enables you to programmatically manage documents, templates, users, and groups. Build powerful integrations to automate your form workflows.

## Base URL

All API requests should be made to:

```
https://api-joy.joyfill.io/v1
```

## API Versions

| Version | Status | Notes |
|---------|--------|-------|
| v1.1 | Current | Latest stable version |
| v1.0 | Supported | Legacy version |

<Note>
  This documentation covers v1.1 of the Joyfill API. For v1.0 documentation, visit [docs.joyfill.io](https://docs.joyfill.io).
</Note>

## Quick Start

<Steps>
  <Step title="Get API Credentials">
    Sign up at [joyfill.io](https://joyfill.io) and get your API access token from the dashboard.
  </Step>
  
  <Step title="Make Your First Request">
    Test your credentials with a simple API call:
    ```bash
    curl https://api-joy.joyfill.io/v1/documents \
      -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
    ```
  </Step>
  
  <Step title="Explore Endpoints">
    Browse our comprehensive API reference to discover all available endpoints.
  </Step>
</Steps>

## Core Concepts

### Documents

Documents are instances of forms filled out by users. Each document can be in different states (draft, submitted, completed) and contains field values.

### Templates

Templates define the structure and fields of your forms. Create documents from templates to maintain consistency across your forms.

### Groups

Groups organize documents and control access permissions. Use groups to manage different projects or teams.

### Users

Users can create, edit, and view documents based on their permissions. Manage user access through the API.

## Authentication

All API requests require authentication using a Bearer token in the Authorization header:

```bash
Authorization: Bearer YOUR_ACCESS_TOKEN
```

Learn more in the [Authentication](/api/authentication) guide.

## Rate Limits

- **Standard**: 1000 requests per hour
- **Enterprise**: Custom limits available

  Contact support@joyfill.io for increased rate limits or enterprise pricing.
