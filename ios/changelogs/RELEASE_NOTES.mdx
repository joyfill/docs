## Release: 3.0.0-rc4
url: https://github.com/joyfill/components-swift/releases/tag/3.0.0-rc4
published_at: 2025-10-10T13:05:17Z
Body:
## What's Changed
**Fixes**
	•	Fixed: Table/Collection image cell triggering onChange twice.
	•	Fixed: Date field calling onFocus before the onChange event.
	•	Fixed: Memory leak and retain cycle between DocumentEditor and JoyfillDocContext.
	•	Fixed: Block field appearance when color value is empty.
	•	Fixed: Issue where filtering number columns with 0 returned empty rows.
        •	Fixed: Including timezone (tz) property in changelog when updating date columns in tables to ensure accurate time handling.
**Read-only Mode Enhancements**
	•	Disabled: Editing of text and barcode column cells in Table/Collection when in read-only mode.
	•	Disabled: Duplicate page functionality in read-only mode.
	•	Disabled: All event handlers while in read-only mode to prevent unintended interactions.
**UI / UX Improvements**
	•	Adjusted: Table/Collection colors for proper appearance in dark mode.
	•	Added: Scrolling support in table column titles to ensure visibility of content when titles are very large.

**Full Changelog**: https://github.com/joyfill/components-swift/compare/3.0.0-rc3...3.0.0-rc4
