# Kotlin SDK Overview

Welcome to the Joyfill Kotlin SDK documentation. Our SDK enables you to integrate powerful, customizable form experiences directly into your Android applications.

## Key Features

  <Card title="Easy Integration" icon="puzzle-piece">

  </Card>
  <Card title="Customizable UI" icon="palette">
    Fully customizable components to match your brand
  </Card>
  <Card title="Multiple Modes" icon="toggle-on">
    Support for readonly, fill, and edit modes
  </Card>
  <Card title="Event Handling" icon="bolt">

- Android 5.0 (API level 21) or later
- Kotlin 1.8 or later
- Android Studio Arctic Fox or later
